## Frequently Asked Questions

**The app isn't detecting my hand.**
Make sure you've granted camera access in System Settings → Privacy & Security → Camera.

**What signs does ASLive recognize?**
ASLive currently recognizes the ASL alphabet (A–Z) and common gestures.

**Where is my camera data stored?**
All processing happens on-device. Your camera feed never leaves your Mac.
